2.解题方法

①原子变量

  • 使用一个原子变量控制，把并发调用变成按顺序调用。

②volatile 关键字

  • 使用 volatile 关键字保证变量的可见性，把并发调用变成按顺序调用。

③信号量

  • 三个线程第一次执行时，因为 flag1 信号量和 flag2 信号量的计数器初始值都为 0，所以需要等 first 线程中释放了 flag1 信号量，second 线程才能开始执行，需要等 second 线程中释放了 flag2 信号量，third 线程才能开始执行。
